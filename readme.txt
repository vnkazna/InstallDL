1. Отключить UAC
2. Настроить Интернет
3. Установить пароль "Password!" для учетной записи, из-под которой будет устанавливаться СЗИ
4. Хотя бы один раз установить вручную Dallas, если не устанавливался, чтобы добавить идателя OOO Confident в доверенные (все из-за DLEZ-9783)
5. Удалить Dallas Lock и перезагрузить
6. Добавить скрипт в %USERPROFILE%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\1.bat
7. Сделать снапшот текущего состояния после вышеперечисленных действий
